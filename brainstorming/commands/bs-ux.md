# bs-ux — UX Flows

Traduce requisiti in flussi utente, schermate e wireframe testuali.

## Workflow

1. Leggi `brainstorm/04-mvp-scope.md` e `02-problem-framing.md`
2. Crea user journey per ogni Must-have
3. Elenca schermate con stati (default/empty/loading/error/success)
4. Crea wireframe testuali ASCII
5. Scrivi `brainstorm/05-ux-flows.md` e `brainstorm/ux/wireframes.md`

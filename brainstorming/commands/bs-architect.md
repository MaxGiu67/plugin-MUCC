# bs-architect — Architettura Tecnica

Definisce stack, architettura, schema dati, API contract e ADR per l'MVP.

## Workflow

1. Leggi `brainstorm/04-mvp-scope.md` e `05-ux-flows.md`
2. Definisci stack tecnologico (FE/BE/DB/Auth/Hosting)
3. Disegna schema dati (entità, relazioni)
4. Definisci API contract (endpoint, method, auth, req/res)
5. Scrivi ADR per scelte chiave
6. Scrivi `brainstorm/06-architecture.md`

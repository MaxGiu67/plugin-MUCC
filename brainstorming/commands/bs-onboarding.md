# bs-onboarding — Onboarding Repo

Orchestrazione dei 5 agenti per onboarding su repository esistente.

## Workflow

1. Leggi `brainstorm/00-assessment.md` per capire quali agenti attivare
2. Codebase Cartographer → `onboarding/codebase-map.md`
3. Dependency Auditor → `onboarding/dependency-audit.md`
4. Doc Writer → `onboarding/developer-docs.md`
5. Bug Triage (se richiesto) → `onboarding/bug-triage.md`
6. Refactoring Coach (se richiesto) → `onboarding/refactor-plan.md`
